创建对象的三种方式

1、利用对象字面量创建对象 ｛｝
2、利用new Object 创建对象
3、利用构造函数创建对象

利用构造函数创建对象

我们为什么需要使用构造函数？
就是因为我们前面两种创建对象的方式一次只能创建一个对象
如果我有10个对象就要复制10份来创建10个对象，
这样很麻烦，有没有更方便的方法呢？
对了，能不能把这个 创建对象的代码 封装成 函数呢？
这样通过这个 函数 来 实现大量代码的重复使用。
因为我们每一次创建一个对象时，每个对象里面都有 很多的属性 和 方法是大量相同的，
因此我们把 创建对象时常用到的 一些相同的属性和方法 抽象出来封装到函数里面
因此我们可以利用函数的方法 重复这些相同的代码 我们就把这个函数 称为构造函数。
又因为这个函数不一样 ，里面封装的不是普通代码，而是对象。
构造函数 就是把 我们对象里面 一些相同的属性和方法 抽象出来封装到函数里面
普通函数里面封装的是普通代码
构造函数里面封装的是对象
